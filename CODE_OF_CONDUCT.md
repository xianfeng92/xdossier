# Code of Conduct

This project adopts the **Contributor Covenant 2.1** as its Code of Conduct.

Read the full text here: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## How to report

If you experience or witness behavior that violates the Code of Conduct, please report it by emailing the maintainer at `<MAINTAINER_EMAIL>` (replace with the actual address before publishing).

All reports will be reviewed and investigated promptly and confidentially. The maintainers are obligated to respect the privacy of anyone reporting an incident.

## Scope

This Code of Conduct applies in all project spaces — issues, pull requests, discussions, code review comments — and also when an individual is officially representing the project in public spaces.

## Enforcement

Maintainers will follow the [Contributor Covenant Enforcement Ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) when responding to reports: from a private written reminder, through temporary ban, to permanent ban for repeated or severe violations.
